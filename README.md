# آپشن یار — اپلیکیشن بازار آپشن و بورس کالای ایران

این پروژه یک اسکلت کامل و قابل‌اجرا از اپلیکیشن اندروید (Kotlin + Jetpack Compose) است که شامل:

- ✅ ورود با جیمیل (Google Sign-In از طریق Firebase) — هر جیمیل یک اکانت کاملا خصوصی و جدا می‌سازد
- ✅ بازار آپشن بورس ایران — رایگان — قیمت زنده + رتبه‌بندی به‌صرفه‌ترین قراردادها با استراتژی کاورد کال
- ✅ بازار بورس کالا — فقط برای مشترکین، با قفل و دکمه خرید اشتراک
- ✅ صفحه خرید اشتراک: نمایش شماره کارت، آپلود عکس رسید، پیگیری وضعیت تایید
- ✅ نمودار سود و زیان استراتژی کاورد کال
- ✅ تاریخچه فعالیت که روی خود گوشی ذخیره می‌شود
- ✅ یک GitHub Action که خودش APK را می‌سازد

## نکته مهم

داده بازار آپشن و بورس کالا در حال حاضر Mock (نمونه با نوسان تصادفی) است، چون بورس ایران API عمومی و مستندی برای داده زنده ندارد. جای این بخش کاملا مشخص و جدا شده (MarketRepository.kt) تا بعدا با یک منبع داده واقعی جایگزین شود.

تایید رسید بانکی در نسخه اول دستی است: ادمین در پنل فایربیس با یک کلیک تایید می‌کند و همان لحظه اشتراک فعال می‌شود.

## گرفتن فایل APK نصبی

بعد از هر push به شاخه main، گیت‌هاب خودکار اپ را می‌سازد:

1. تب Actions ریپازیتوری را باز کن
2. آخرین اجرای «Build Debug APK» را باز کن و صبر کن سبز بشود
3. پایین صفحه، بخش Artifacts را باز کن و فایل covered-call-app-debug-apk را دانلود کن
4. فایل zip را با فایل‌منیجر باز کن، APK داخلش را نصب کن

## راه‌اندازی فایربیس (برای ورود با جیمیل و دیتابیس)

1. برو به console.firebase.google.com و یک پروژه بساز
2. Authentication را فعال کن (روش Google)
3. Firestore Database بساز
4. Storage را فعال کن
5. یک اپ اندروید با package name دقیقا `com.marghazari.coveredcall` اضافه کن
6. فایل google-services.json را دانلود و داخل پوشه app/ آپلود کن
7. مقدار Web client ID را در app/src/main/res/values/strings.xml جایگزین default_web_client_id کن

## تایید رسید اشتراک

هر بار کاربری رسید آپلود کند، سندی در Firestore زیر users/{uid}/receipts/{receiptId} با وضعیت PENDING ساخته می‌شود. برای تایید، status را به APPROVED تغییر بده و در سند users/{uid}، فیلدهای isSubscribed=true و subscriptionExpiryMillis (الان + 30 روز به میلی‌ثانیه) را ست کن.
